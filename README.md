📘 MathBuddy AI

MathBuddy AI is an AI-powered mathematics learning platform for KG to Class 5 (NCERT-based) that provides step-by-step explanations in a safe, structured environment.

🌐 Live Demo

🚀 Try it here:
👉 https://mathbuddy-ai.netlify.app/

🚀 Features
📚 NCERT-based structured learning
🤖 AI-powered doubt solving
🎯 Personalized explanations
📺 Topic-based references
📅 Study timetable generator
🛡️ Safe, controlled AI responses
🧠 Tech Stack

Frontend

React.js
Next.js
Tailwind CSS

Backend

Node.js
Express.js

AI

OpenAI API

⚙️ Setup
git clone https://github.com/your-username/mathbuddy-ai.git
cd mathbuddy-ai
Install & Run
cd frontend
npm install
npm run dev
cd backend
npm install
npm start

User selects class & topic
Sends question
Backend adds context
AI generates step-by-step answer
